Affinché un sito web sia accessibile online, necessita di un dominio (un nome univoco) e un indirizzo IP. Ad esempio, https://google.com è un dominio che punta ad un server di Google tramite un indirizzo IP. ovvero 172.217.20.206se si prova a digitare l'ip sulla barra di ricerca comparirà il sito di google
