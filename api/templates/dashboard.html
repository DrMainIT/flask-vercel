Come funziona un sito web

Un sito web è diviso in tre parti principali:

  • Frontend: Gli elementi visivi (pulsanti, testo, grafiche) con cui gli utenti interagiscono.
  • Backend: La logica che elabora i dati dal frontend e fornisce risposte.
  • Database: Una raccolta di dati organizzati per memorizzare e recuperare informazioni.

Frontend

Il frontend include tutti i componenti visivi che gli utenti vedono e con cui interagiscono, come ad esempio:

  • Pulsanti



  • Form:



  • Testo e immagini

Backend

Il backend gestisce la logica del sito. Elabora i dati dal frontend ed esegue azioni come:

  • login di un sito
  • Interagire con il database per recuperare o salvare informazioni
  • elaborare i dati

Alcuni framework comuni per il backend includono:

  • Flask (usato in questo progetto)
  • Django
  • FastAPI
  • Express.js
  • Spring

Database

Un database viene utilizzato per memorizzare e recuperare dati, come i dettagli dell'utente o l'inventario dei prodotti. In questo progetto, usiamo SQLite, che è leggero e facile da configurare.

Collegare tutto

Il backend collega il frontend e il database. Ad esempio:

Un utente compila un form per registrarsi. Il backend elabora i dati e li salva nel database.

Richieste HTTP

La comunicazione tra il frontend e il backend avviene tramite richieste HTTP. Le più comuni sono:

  • GET: Recupera dati
  • POST: Invia dati
  • PUT: Aggiorna dati
  • DELETE: Rimuove dati

2. Route e la loro funzionalità

Una route è fondamentalmente un percorso verso un file specifico o un blocco di codice che restituisce qualcosa in questo caso quando si accede alla route ‘/’ verrà visualizzato “hello world”

@app.get('/')
def index():
    return 'Hello, World!'

Jinja

Jinja è un motore di template scritto in Python. Viene utilizzato da framework per applicazioni web come Flask, è molto utile per passare dei dati al frontend. Ipotizziamo di avere una route che oltre a passare una pagine html all'utente con la funzione render_template passa anche un titolo

@app.route('/')
def index():
    return render_template('index.html', title='Home')

per visualizzare il titolo bastera nella nostra pagina html usare questa semplice e minimale sintassi

<h1>{{ '{{ title }}' }}</h1>

Cicli in jinja

potrebbe essere utile a volte passare delle liste piuttosto che variabili come una lista di persone. In questo caso per scrivere un ciclo in jinja basterà digitare

<ul>
    {{'{% for person in people %}'}}
        <li>{{ '{{ person }}' }}</li>
    {{'{% endfor %}'}}
</ul>

per ulteriori informazioni visitare la documentazione https://jinja.palletsprojects.com/en/stable/
